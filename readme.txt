== Description ==

Update Stat displays the daily statistics updates of Russian and Ukrainian search systems Google and Yandex and indicators CY and PR. 

== Installation ==

To use Update Stat, you will need:

* 	an installed and configured copy of WordPress version 3.4.x

*	FTP or SFTP access to your web host

= New Installations =

1.	Download the Update Stat archive in zip or gzipped tar format and
	extract the files on your computer. 

2.	Create a new directory named `update-stat` in the `wp-content/plugins`
	directory of your WordPress installation. Use an FTP or SFTP client to
	upload the contents of your Update Stat archive to the new directory
	that you just created on your web host.

3.	Log in to the WordPress Dashboard and activate the Update Stat plugin.

5.	Configure the plugin and add it to the plugin settings page.
